1. 第一题
```javascript
let a = {n : 1};
let b = a;
a.x = a = {n: 2};

console.log(a.x);
console.log(b.x);
```
> 这个题目主要考察赋值语句中变量取值问题。
正常的思路，赋值语句先执行`a= {n:2}`,然后再把这个`{n: 2}`赋值给`a.x`,问题是这个`a.x`的`a`指向谁？
注意这里的a不是指语句中赋值之后的`a`，而是在执行赋值语句之前的`a`，也就是上面与`a`有相同引用的`b`。所以打印出来的答案
是： `undefined` `{n: 2}`
